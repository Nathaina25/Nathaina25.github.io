Description of The Project

NileDotCom is an e-commerce application that allows users to browse and purchase products online. The application features a user-friendly GUI, a dynamic product catalog, shopping cart functionality, and a secure checkout process.

It is built using Java and incorporates file handling for robust order management. I completed this project as part of the Enterprise Computing Course at UCF.

Skills Learned

  • Testing and Debugging: Identifying and fixing logical errors in Java.
  • Problem Solving: Designing the workflow for an online checkout system.
  • Technical Communication: Explaining system requirements and features.
  • Web Management Development: Understanding the back-end logic of e-commerce.
